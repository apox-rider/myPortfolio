<h2>‚öôÔ∏è Setup & Installation</h2>
<h3>1. Clone the repository</h3>
<p>Bashgit clone https://github.com/your-username/your-portfolio.git
cd your-portfolio</p>
<h3>2. Install dependencies</h3>

Bashnpm install
# or
yarn install
# or
pnpm install

<h3>3. Set up EmailJS (for contact form)</h3>
<summary>
Go to https://www.emailjs.com/ and create a free account
Add your email service (Gmail, Outlook, etc.)
Create an email template
Copy your Service ID, Template ID, and Public Key
</summary>

<h3>4. Create .env.local file in the root</h3>
<summary>
envNEXT_PUBLIC_EMAILJS_SERVICE_ID=your_service_id
NEXT_PUBLIC_EMAILJS_TEMPLATE_ID=your_template_id
NEXT_PUBLIC_EMAILJS_PUBLIC_KEY=your_public_key
</summary>
<h3>5. Run locally</h3>

Bashnpm run dev
# or
yarn dev
# or
pnpm dev
Open http://localhost:3000 to view it.

<h3>üöÄ Deployment</h3>
<p>The easiest way to deploy is with Vercel:</p>

Push your code to GitHub
Go to vercel.com

Add the same environment variables from .env.local in Vercel dashboard
Deploy!
